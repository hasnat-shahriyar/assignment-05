1. What is DOM and what is the purpose of DOM?

Ans : DOM stands for Document Object Model.
It is used to create and transform static websites designed with HTML and CSS into dynamic and interactive web pages.

2. How will you select HTML elements using DOM? Name the DOM methods.

Ans :
getElementById(): To select an element by its ID attribute.
getElementsByClassName(): To select all elements with the specified class name.
getElementsByTagName(): To select all elements with the specified tag name.
querySelector(): To select the first element that matches the specified CSS selector.
querySelectorAll(): To select all elements that match the specified CSS selector.

3. What is event bubble?

Ans :
Event bubbling is a way events work on a web page.
It means when something happens (like a click),
the web page checks how to deal with it starting from where it happened and then moving up to the main part of the page.

4. What is a callback function and why will you use it?

Ans :
A callback function is a function that you pass as an ingredient to another function. This other function, like a chef, uses this ingredient (the callback) at a specific point while it's doing its work.

Callback functions are quite handy in two main situations:

Asynchronous Operations: Imagine you're cooking, and one step requires the rice to be ready before you can stir-fry it. So, you ask someone to keep an eye on the rice and tell you when it's done. That someone is like your callback function. In coding, this is used for tasks that take some time, like waiting for data from the internet or a file.

Event Handling: Consider a button you click. You want something to happen when you click it, like music starting to play. The button can't directly control the music, but it can say, "Hey, when you click me, tell the music player to start playing." That message is a callback function.

In simpler terms, callbacks are like special instructions you give to a function, telling it what to do when a specific condition is met, like "call this function when the rice is ready" or "play music when the button is clicked."
